Get Started

The Darwin RESTful API is a community driven project supported by the HackTrain team - the code is open source, and available to any Trainhacker. You can get started by visiting the documentation, or jumping straight to the code.




Big Data, Darwin, and the Railway Industry.

Darwin is one of the most important datafeeds in the railway industry. It powers uncountable train operating companies and railway service providers, and allows them to deliver high quality train services that get us from A to B every day.



Innovating. One Dataset at a time.

Given the importance of this datafeed, the HackTrain kick-started the Darwin RESTful project - an unofficial Darwin REST interface supported by the HackTrain team. Our vision with this project is to empower trainhackers to build unique, revolutionary projects that will revolutionize the train industry.






Built by the community, for the community.

The Darwin RESTful API is a community driven project - the code is open source, and available for any Trainhacker. Pull requests are more than welcome. We encourage trainhackers to contribute, as we look for this project to become as tailored as possible for the community itself.
